# mattis

The decoder for pingy's verification service.

```
./mattis
```

```
The decoder for pingy's verification service.

Usage:
  mattis [flags]
  mattis [command]

Available Commands:
  decode      Decode a specific multisig transaction.
  version     Print the version information for this command line tool.

Flags:
  -h, --help   help for mattis

Use "mattis [command] --help" for more information about a command.
```
